# Performance Reports - Time Report Notes

## Group By Empty Groups

When Group By is active, Net Net renders only groups that contain at least one visible filtered time entry. Empty groups are never shown.

If no entries match the current filters, the report shows a single filtered-empty state instead of blank group sections.

## Duration Display And Editing

Time Report supports Decimal and Clock duration display. Decimal is the default.

Every row shows the logged duration clearly in the Duration column. Timer entries show time in / time out in the Time column plus the duration. Manual entries show Manual in the Time column plus the duration.

Owner and Admin users can edit Duration inline from the report. Timer entries with time in / time out can also edit those times inline, and the saved duration recalculates from the updated timer range.

## Report Roles

Time Report answers individual time entry, note, and cleanup questions.

Team Report answers assigned effort, logged effort, due work, and on-time completion by Team Member.

Team Dashboard answers forward-looking capacity and workload questions.
